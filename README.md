# Word Theme

Word theme brings Microsoft Word to Visual Studio Code.

![Word for Visual Studio Code](https://raw.githubusercontent.com/hbisneto/hbisneto.github.io/main/word-theme/banner.png)

## Available Themes

- Word Theme (Dark)
- Word Theme (Light)
